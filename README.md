# Yo! Sushi Operations Analytics

## Project Overview

This project aims to build an end-to-end restaurant operations analytics system using:

- Excel
- PostgreSQL
- SQL
- Python
- pandas
- matplotlib
- Power BI
- DAX

The goal is to analyse restaurant operations including:

- sales performance
- staffing efficiency
- waste management
- operational KPIs
- business trends

The project simulates a real business intelligence workflow used in operational analytics environments.

---

# Planned Analytics Areas

## Sales Analytics

- busiest days
- busiest hours
- revenue trends
- top-selling items

## Waste Analytics

- waste cost analysis
- high-waste items
- waste trends

## Staffing Analytics

- staff efficiency
- labour utilisation
- shift performance

## Operational KPIs

- revenue per staff member
- waste percentage
- peak-hour analysis
